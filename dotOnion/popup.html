garlic is a tor proxy handler that can make accessing your favorite .onion sites through your favorite browser simple - however, you should not expect added security/privacy from using it.

If you are using garlic for privacy, I highly recommend getting the Tor Browser instead. It is a lot more secure as it does other stuff to protect your privacy, garlic is only intended for letting you access tor sites, it doesn't add any security layers.
