# garlic  
A browser extension enabling you to view **.onion** addresses in your favorite Chromium-based browsers (**Chromium**, **Chrome**, **Brave**, **Edge**, **Vivaldi** *et al*). 

I plan to port the extension over to Firefox soon, as well.
